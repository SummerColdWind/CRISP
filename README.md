# CRISP: Corneal confocal microscopy Real-time Image Stitching Pipeline

CRISP is a plug-and-play real-time image stitching project for corneal confocal
microscopy examination video streams. It is designed to generate incremental
CCM mosaics without requiring additional hardware or extra operator actions.

## Source Code Availability

The project source code will be made public after the relevant patent
application has been formally accepted.
